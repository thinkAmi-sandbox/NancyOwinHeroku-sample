## Tested environment

 * Windows7 x64
 * .NET Framework 4.5
 * Heroku Toolbelt 3.11.1
 * NuGet package
    * Microsoft.Owin.Hosting 3.0.0
    * Microsoft.Owin.Host.HttpListener 3.0.0
    * Nancy.Owin 0.23.2
